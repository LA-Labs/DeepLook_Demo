# LookKit_Demo
Demo project for LookKit SDK

# Introduction
This LookKit_Demo is design for you to gain a basic understanding of the LookKit Mobile SDK. 
It will implement the Face clustering view functionality.

# Requirements
- iOS 13.0+
- Xcode 12.0+
- Swift 5.3+
