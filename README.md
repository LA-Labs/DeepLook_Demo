# LookKit SDK Demo

# Introduction
This LookKit Demo is design for you to gain a basic understanding of the LookKit Mobile SDK. 
The sample App shows scenarios such as Object Detection, Face Verification, and Face Grouping.

# Requirements
- iOS 13.0+
- Xcode 12.0+
- Swift 5.3+
